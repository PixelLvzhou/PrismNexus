### 实现计划

1. **安装地图库**：安装 Leaflet.js 及其类型定义，这是一个轻量级的开源地图库，适合简单地图浏览功能
2. **创建地图组件**：在 `src/components` 目录下创建 `Map.vue` 组件
3. **配置地图组件**：
   - 引入 Leaflet 库和样式
   - 初始化地图实例，设置中心点为中国
   - 配置地图图层和缩放级别
   - 添加基本地图控件（缩放控件、比例尺等）
4. **在首页引入地图组件**：在 `src/views/Home.vue` 中，将地图组件添加到轮播组件下方
5. **样式调整**：
   - 设置地图容器的高度和宽度
   - 适配响应式设计
   - 调整地图组件与其他组件的间距
6. **测试运行**：确保地图正常加载和显示

### 技术选择
- **地图库**：Leaflet.js（轻量级、开源、易于集成）
- **地图数据源**：OpenStreetMap（免费开源的地图数据）
- **实现方式**：Vue 3 Composition API

### 预期效果
- 地图模块显示在中国地图范围内
- 支持鼠标拖拽、滚轮缩放
- 显示缩放控件和比例尺
- 响应式设计，适配不同屏幕尺寸
- 位于轮播组件下方，形成合理的页面布局
