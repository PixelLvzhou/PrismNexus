# PrismNexus

现代化前端应用框架。

## ✨ 功能特性

- 用户认证与权限管理
- 主题切换
- 动画效果与交互反馈

## 🛠️ 技术栈

- Vue 3
- TypeScript
- Vite
- TailwindCSS
- Vue Router
- Axios

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 📜 可用脚本

| 脚本 | 描述 |
|------|------|
| `dev` | 启动开发服务器 |
| `build` | 构建生产版本 |
| `preview` | 预览构建结果 |
